# rrmqr - Find and read QR-Codes and rMQR-Codes

This crates exports functions and types that can be used to search for QR-Codes
and rMQR-Codes in images and decode them.

QR code is from rqrr, rMQR code was added on 2026 Jan-Mar.
rMQR code needs improvements for finding the rMQR code's corners robustly.
This library drifted away from the rqrr crate so it is not a fork anymore.
